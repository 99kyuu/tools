# python-tools
打算利用python来实现各种小工具(玩具)

**web_scanner.py**

> 这个脚本是用字典跑网站路由的, 目的是发现一些曾经爆出过漏洞的隐藏链接, 比如后台登陆入口, 备份的配置文件等;

**web_runtime.py**

> 这个脚本运行后, 可以在浏览器页面上运行python代码片段;

**mail_client.py**

> 这个脚本是用tkinter模块写的简易邮件发送客户端;

**restart_windows.py**

> 预约一个windows定时关机任务, 取消定时关机任务

**douban_comment.ipynb**

> 抓取豆瓣电影短评, 结巴分词, 生成词云
